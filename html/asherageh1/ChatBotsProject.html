Chat-Bots

A Brief Overview

My interest in chat-bots originated when I first started creating Simon. I started researching and learning about Chat-bots with Codecademy. This taught me that there were three major types of bots: Rule Based, Retrieval Based, and Generative.

Rule Based chat-bots act how they sound. They have a set of pre-defined rules (such as questions and answers) that it cannot deviate from. To expand, these bots cannot generate their own answers but are given a set of questions that it can ask along with answers that it understands. These bots can be seen in places like with a customer support agent. Siri is a good example of a retrieval based chat-bot. See here to learn more

Retrieval based chat-bots are more complex than simple rule based chat-bots and delve into the world of Natural Language Processing. They are given a database of predefined responses and the program tries to figure out the best response when the user gives a statement. See here to learn more

Generative chat-bots are the most complex type of conversational AI. As the name implies, generative chat-bots are more advanced that retrieval based bots in the sense that they can generate text instead of retrieving it from a database. See here to learn more
